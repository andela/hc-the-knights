Frequently Asked Questions

Healthchecks is a platform that allows the user to setup cron jobs and keep track of their status.

What is a cron job?

A cron job is a command which has been scheduled to run automatically at a specified time and date.

How do you create a cron job?

To create a cron job, you have to log in to healthchecks



On successful login, you are redirected to the checks page where you can add, pause or remove a check. When adding a check you can set the timer for the period and grace time from 1 minute to up to 2 months with an option of minutes, hours, days, weeks or months


What do the icons on the checks symbolise?

Each of the different icons is used to indicate when the check is in a different state

A check can be in one of these states:

New. A check that has been created, but has not received any pings yet.
Monitoring Paused. You can resume monitoring of a paused check by pinging it.
Up. Time since last ping has not exceeded Period.
Late. Time since last ping has exceeded Period, but has not yet exceeded Period + Grace.
Down. Time since last ping has exceeded Period + Grace. When check goes from "Late" to "Down", healthchecks.io sends you an alert.
Nag User. Time since last Period + Grace elapsed. When check goes from "Late" to "Down", healthchecks.io sends you an alert. If you don't resolve the issue for the check to go up, it sends another alert after specified interval Period + Grace.

How do you run a cron job?

Once a cron job is created, hover over the job url. You will notice the word copy which is a hyperlink. click on it. This copies the url and you can then paste that url in your browser and run it. This starts the job



You can also view the log of a check, the various usage options, pause as well as delete a check



How do you add integrations?

Go to the integrations page and follow the available guidelines to setup the integrations. Different integrations have different setup instructions
